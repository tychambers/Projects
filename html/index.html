Flask Movie API Site

Below is a link to the repository for the source code of a website I built. The website uses Python's Flask as a framework for the site. The site has an authentication system which lets you create an account and login. After logging in you are given a key to use the API. The API has a database of 10,000+ movies which I imported from a dataset at kaggle.com. I used jupyter notebooks and pandas library to format the data and import it into the site's database. The API then queries the database to retrieve the results. I also used the website to show documentation on how to use API. There are 2 main queries: get information about a movie from a name input, and get the movies that an actor is in through using the actor's name as an input. I also created a DockerFile for the project so it can easily be containerized.
